Fetch and parse Aikatsu! DCD mypage data.

# dcdkatsu command usage

## User
`dcdkatsu user <User ID>`

## Digital Binder
`dcdkatsu digital_binder <User ID> <Series ID>`

## Card
`dcdkatsu card <Card URL>`
